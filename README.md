# odin-rock-paper-scissors

## Overview

Odin Rock Paper Scissors merupakan tugas Project: Rock Paper Scissors pada modul "Javascript Basic - Foundations Course" di [The Odin Project](https://www.theodinproject.com/). Pada project ini, saya akan membuat sebuah website game "Gunting Batu Kertas (ID)" yang akan membuat Player bermain melawan Computer.

## Fitur

Website ini akan menampilkan fitur:

- Score kemenangan Player dan Computer.
- Icon "Rock, Paper, Scissors".
- Pilihan "Rock, Paper, Scissors" untuk Player.
- Pop-up animasi menang, kalah, atau seri.

## Catatan

Website ini masih dalam tahap pengembangan sampai saya menyelesaikan course dari The Odin Project. Sampai saat ini, website ini masih belum responsive untuk layar yang lebih kecil seperti tab dan mobile.
